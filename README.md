# Projeto: Livraria (PHP + MySQL)

Este projeto usa XAMPP (Apache + MySQL) e pode ser editado no Sublime Text.
Contém dois CRUDs relacionados: **Autores** e **Livros** (cada livro pertence a um autor).

## Como rodar
1. Instale o XAMPP e inicie Apache e MySQL.
2. Copie a pasta `livraria` para `C:\xampp\htdocs\` (Windows) ou `/opt/lampp/htdocs/` (Linux).
3. Abra `http://localhost/livraria/` no navegador.
4. Importe `database.sql` via phpMyAdmin ou execute no MySQL.
5. Edite `inc/db.php` se necessário (usuário/senha do DB).
